# Airtable Schema

Create a table with the following fields.

| Field Name | Type |
|-----------|------|
| Job Title | Single line text |
| Company Name | Single line text |
| Location | Single line text |
| Time Posted (Days) | Number |
| Employment Type | Single line text |
| Industries | Single line text |
| Apply URL | URL |
| Job Description | Long text |
| Easy Apply | Checkbox |

## Notes
- Use Apply URL as unique identifier
- Avoid formula fields for insert targets
